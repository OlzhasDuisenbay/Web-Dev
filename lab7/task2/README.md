# OOP Project - Animal
## Description
This project demonstrates Object-Oriented Programming (OOP) in Python.
We created a base class `Animal` and two child classes `Dog` and `Cat`.

## Features
- Inheritance
- Method overriding
- Modules (models.py, main.py)

## Classes

### Animal
- Attributes: name, age, color
- Methods: eat(), sleep(), speak()

### Dog
- Inherits from Animal
- Additional attribute: breed
- Methods: fetch()
- Overrides speak()

### Cat
- Inherits from Animal
- Additional attribute: lives
- Methods: climb()
- Overrides speak()
